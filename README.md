# Umamusume Birthday Calendars (ICS)
This repository publishes **subscription-ready** iCalendar (`.ics`) files for Umamusume characters.

## Calendars
- Japanese (JA): `umamusume_birthdays_ja.ics`
- English (EN): `umamusume_birthdays_en.ics`
- Simplified Chinese (ZH-CN): `umamusume_birthdays_zh-cn.ics`
- Traditional Chinese (ZH-TW): `umamusume_birthdays_zh-tw.ics`

## Scope
This calendar includes birthdays of characters categorized under "Umamusume".
It does not include supporting, anime-exclusive, or other spin-off categories.
Both playable and not-yet-playable Umamusume characters are included when birthday information is available.

## Data Sources
Birthday and name information are aggregated from multiple publicly available fan-maintained resources.
This repository only redistributes processed calendar files (`.ics`) and does not claim ownership of the original data.

## Update Policy
Calendar files are updated periodically.
Updates are typically performed after:
- New character additions
- Birthday data corrections
- Name mapping updates

## Notes / Data Freshness
- Character count may differ across data sources.
- Birth year is not guaranteed for newly added characters, because it is derived from sources that may update slower than others.
